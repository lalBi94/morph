# Morph - Eyesing [FR]
Morph est un projet inspiré par le regretté Supervive. L’objectif est de redonner vie à des mécaniques de jeu très appréciées par sa communauté, tout en proposant une expérience originale. Morph n’est pas une suite officielle et est distribué sous licence MIT, vous laissant libre de le modifier à votre guise.

## Gameplay
Morph est un battle royale 3D dynamique pour 20 joueurs, où le but est simple : être le dernier survivant. La caméra est de type MOBA, et chaque personnage dispose de sorts uniques qui influencent la stratégie de combat. Côté graphismes, le style choisi est cell-shading low poly, pour un rendu à la fois clair, lisible et moderne.

## Informations techniques
Le moteur utilisé est Godot 4.5, le serveur est développé en Rust, et le launcher repose sur React.js avec une WebView Electron.js.
La communication se fait via UDP pour le jeu et WebSocket pour le launcher. Cette architecture assure à la fois performance, rapidité et flexibilité.

# Morph - Eyesing [US-EN]
Morph is a project inspired by the late Supervive. Its goal is to revive gameplay mechanics highly appreciated by its community while offering an original experience. Morph is not an official sequel and is distributed under the MIT license, allowing you to modify it freely.

## Gameplay
Morph is a dynamic 3D battle royale for 20 players, where the objective is simple: be the last survivor. The camera is of the MOBA type, and each character has unique spells that influence combat strategy. Visually, the game adopts a low-poly cell-shading style, resulting in a clear, readable, and modern look.

## Technical Information
The engine used is Godot 4.5, the server is developed in Rust, and the launcher is built with React.js using an Electron.js WebView. Communication is handled via UDP for the game and WebSocket for the launcher. This architecture ensures performance, speed, and flexibility.
